Express 사용 안하고 API 서버 만들어보기

Delryn's Blog

DB 연결은 안 하고 기타 라이브러리 없이 기본적인 구조 및 라우팅만 만들어 보기로 했다.

POST의 경우에는 공식 문서 가이드에 따라 해봤는데 잘 넘어오는 걸 확인 할 수 있었다.
