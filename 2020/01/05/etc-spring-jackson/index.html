Spring에서 json 다루기

delryn's blog
2020-01-05
Spring

의존성

pom.xml(maven)에 jackson을 추가 한다.

위와 같은 json을 다루기 위해서 형태?에 맞는 데이터 클래스를 작성해줘야 한다.

getter/setter가 하나 라도 생략 시 jackson 오류가 나므로 체크를 잘 하자.

예시로 User라는 클래스를 작성해본다. 솔직히 자바스크립트에 비해 좀 귀찬다..

GameData 클래스.

UserLocale 클래스

데이터 모델 클래스가 완료되었으니 컨트롤러에서 써보자.

의도한 대로 나온다. 이제 GET, POST로 오는 데이터를 처리 해보자.

GET

get인 경우 쿼리스트링의 데이터를 받아야 하는데 당연히 스트링이므로 json 파싱을 해줘야 한다.

데이터 클래스에 넣어주기만 하면 된다.

POST

post는 파라미터 받는 곳에 데이터 클래스를 적어주니 알아서 들어옴.
